Built with **Next.js**, **TypeScript**, **ShadCN UI**, and **Tailwind CSS**, the application provides a premium SaaS experience with AI-powered startup generation and dashboard management.

# 🎯 Problem Statement

Turning an idea into a startup requires extensive market research, business planning, MVP design, technology selection, and marketing strategy.

Many aspiring founders struggle to organize these tasks efficiently and often spend weeks creating documentation before writing any code.

---

# 💡 Solution

LaunchForge AI enables users to generate complete startup plans using Google Gemini AI.

Within seconds, users receive:

* Startup Summary
* Problem Statement
* Solution
* Business Model
* MVP Features
* Recommended Tech Stack
* Marketing Strategy
* Product Roadmap
* Estimated Budget

All generated startups are automatically saved and managed through a personalized dashboard.

# ⚙️ Installation & Setup

```bash
# Clone repository
git clone YOUR_FRONTEND_REPOSITORY

# Navigate
cd frontend

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local

# Add backend URL
NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1

# Start development server
npm run dev
```
